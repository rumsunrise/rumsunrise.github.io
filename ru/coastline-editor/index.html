Coastline - Text Editor

Простой минималистичный текстовый редактор для Android.

Coastline - это минималистичный текстовый редактор для Android. Приложение предназначено для редактирования простых текстовых файлов (.txt и тому подобные).

Редактор поддерживает только файлы в кодировке "UTF-8". Если вы хотите использовать старые Windows-специфичные кодировки (вроде Windows-1251 или Windows-1252), то приложение вам не подойдет. Просто попробуйте найти другой редактор.

Основные функции и особенности приложения:

  • Чистый приятный дизайн без визуального мусора
  • Возможность настраивать тип и размер шрифта
  • Возможность настраивать межстрочный интервал
  • Автосохранение файла: сохраните файл один раз и все последующие изменения будут сохранены автоматически
  • Встроенный простой проводник для открытия и сохранения файлов
  • Автоматическая подстановка расширения .txt при первом сохранении файла
  • Возможность запоминать последний открытый файл
  • Возможность не выключать подсветку экрана при работе с приложением
  • Шаринг текста в другие приложения с использованием стандартного механизма ОС
  • Поддержка работы на смартфонах и на планшетах

Приложение переведено на следующие языки:

  • Английский
  • Русский
  • Немецкий

Известные проблемы

  • Медленная работа с большими файлами
  • Некорректная работа с ФС Android 7 и новее
  • Иногда возникают трудности с установкой приложения как программы по умолчанию для открытия текстовых файлов (Android 7 и новее)
  • На Android 5 и новее невозможно сохранить файл на внешнюю SD-карточку
